# 🖥️ Rodrigues Tech - Assistência e Montagem de PCs

Site profissional para serviços de assistência técnica, montagem e manutenção de computadores em Jandira/SP.

![Rodrigues Tech Banner](https://via.placeholder.com/1200x400/0A0A0A/00C2FF?text=Rodrigues+Tech)

## 📋 Sobre o Projeto

Site institucional desenvolvido para a Rodrigues Tech, uma empresa especializada em:

- Montagem e manutenção de computadores
- Limpeza técnica e troca de pasta térmica
- Upgrades de hardware
- Diagnóstico e reparo de software
- Consultoria para montagem de PCs

### 🎯 Público-alvo

- Clientes da região de Jandira/SP e proximidades
- Desde usuários domésticos até entusiastas de hardware
- Pessoas que buscam serviços técnicos de qualidade com transparência

## ✨ Funcionalidades

### Design e UX

- 🎨 Design moderno com tema dark/tecnológico
- 📱 Totalmente responsivo (mobile-first)
- 🎭 Animações dinâmicas e interativas
- 🌈 Gradientes e efeitos neon
- 🔍 Scroll suave entre seções

### Animações Implementadas

- Navbar com slideDown ao carregar
- Cards com fadeInUp escalonado
- Ícones com rotação 3D no hover
- Botões com efeito ripple
- Imagem principal flutuante
- Preços com animação pulsante
- Links com underline animado
- Scroll reveal progressivo
- Step numbers com rotação

### Seções do Site

1. **Header/Navbar** - Logo, menu de navegação e botão WhatsApp
2. **Hero Section** - Apresentação principal com CTA
3. **Serviços** - Cards com todos os serviços oferecidos
4. **Preços** - Tabela completa de valores organizada por categoria
5. **Processo** - Fluxo de atendimento passo a passo
6. **Contato** - WhatsApp, redes sociais e informações de localização

## 🛠️ Tecnologias Utilizadas

### Frontend

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização avançada com animações
- **JavaScript** - Interatividade e scroll reveal

### Bibliotecas e Fontes

- **Google Fonts** - Orbitron, Exo 2, Rajdhani, Poppins, Montserrat
- **Font Awesome 6** - Ícones profissionais
- **AOS Library** (opcional) - Animações de scroll

### APIs e Integrações

- **WhatsApp API** - Link direto para conversa
- **Instagram/Facebook/YouTube** - Links para redes sociais (configuráveis)
